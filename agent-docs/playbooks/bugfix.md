# Playbook - Bugfix

## Input minimo

- Sintomo riproducibile.
- Condizioni in cui avviene.
- Comportamento atteso.

## Flusso

1. Riproduci bug.
2. Identifica root cause.
3. Applica fix minimo.
4. Aggiungi test di regressione se fattibile.
5. Esegui loop di verifica e documenta risultato.

## Output atteso

- Root cause chiara.
- Diff minimo e mirato.
- Passi per verificare che il bug sia risolto.
